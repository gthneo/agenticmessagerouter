# AMR 营销资料（受众：销售员）

> 适用版本 **v0.11.0** ｜ 更新日 2026-06-28 ｜ 受众：易链**销售员**（对外卖，非技术）

## 一句话
AMR 是现代互联网人的**沟通中枢**——把微信/电话/飞书等所有渠道的人、对话、事，
归到一个界面，AI 帮你**看懂关系、拟好话术、守住承诺**，但**发不发永远你说了算**。

## 三支柱（卖点话术）
| 支柱 | 客户痛点 | AMR 给的价值 |
|---|---|---|
| **关系智能·人和事谐** | 一个人微信好几个号、还有电话飞书，散在各处记不住、漏联系 | 一人多渠道多号自动归一；该联系谁主动浮出（🔴逾期/⭐关注），不再丢单漏人 |
| **沟通教练·外圆内方** | 话不会说、得罪人、被识破、关键时刻没词 | AI 诊断这段对话的问题(外圆内方+以牙还牙以蜜还蜜)，给一句话口径+3版话术 |
| **养心护体·保健医师** | （路线图）对内情绪/状态管理 | 下一阶段交付 |

## ROI 话术
- **不漏人**：关系账户染色🟢🟡🔴+主动队列，逾期自动提醒 → 减少客户流失/跟进遗漏。
- **会说话**：AI 诊断+多版话术 → 一线人员沟通水平拉平到"老法师"水准，缩短成单周期。
- **守底线**：人在回路、永不自动群发、可解释不黑箱 → 敏感关系零事故，老板放心。
- **离线可用**：断网/无 AI 也能正常收发 → 不被模型绑架，业务连续。

## 同行对比（差异点）
- 不是群发机器：**管的是"通道本身"与"关系"**，按 人×事×风险 选最合适的渠道，不渠道盲发。
- 不是黑箱 AI：每条话术都是**草稿待人审**，诊断结构化可查，方法可配置。

## 适用场景（真实验证）
客户关系危机灭火、深聊跟进、话术被识破修复、没钱时诚实沟通、内圈周期维护、定时调出……
（详见场景语料，均 dogfooding 验证。）

## v0.7 → v0.8 这一版新增了什么（卖点话术 delta）

> 用法：客户已认可三支柱后，用下面 5 句强化"更准、更省心、更像你"的体感。

- **多渠道多号自动找对人**（关系智能加强）：一个客户微信好几个号、又有电话飞书，AMR 自动
  挑那个**现在发得出去**的端点发，号码自动去重，还能**钉住你偏好的那个号**——再不会发到一个
  已经不看的旧号上。话术：「同一个人散在五个口子，系统替你挑当下最通的那个。」
- **人·会话·事三栏，一件事一张卡**：左看人、中看跨渠道合在一起的完整对话、右边一张"事卡"从
  **🩺诊断 → ✨话术 → 发送 → 📌承诺/办结**走完一件事。话术：「不是又一个聊天软件，是把每件事
  办到闭环的工作台。」
- **沟通教练升级·外圆内方 + 以牙还牙以蜜还蜜**（沟通教练支柱）：诊断引擎固化成可复现、可查、
  可配置的结构化诊断，对方释善意就蜜还蜜、对方强硬就有度对等——**不升级、不记仇、对方回头立即
  回暖**。话术：「教一线人员什么时候该圆、什么时候该硬，分寸由方法把关，不靠天分。」
- **认得出"你自己"**（v0.8 新）：你本人的多个微信/电话/飞书身份一次声明，**你发的消息显示为"我"**，
  和自己/分身的对话不会再被当成"该联系的客户"混进队列。话术：「连你自己都认得，名单里只剩真客户。」
- **越用越像你说话**（口吻沉淀）：草稿模仿你自己已发消息的口吻，不是千人一面的客服腔。
  话术：「拟的话术带你的味道，发出去不违和。」

> **三支柱进度**：关系智能、沟通教练本版均加强落地；**养心护体·保健医师仍为下一批交付**
> （对内情绪/状态管理），对外口径不变，不要超卖。

## v0.9 → v0.11.0 新增（卖点话术 catch-up · 销售口径）

> 这一版有两件大事可以**升格卖**：（1）"监管下自动沟通"**第一步已经交付**（但只可讲已交付的那一步，
> 别超卖）；（2）新增"可观测·运维即服务"卖点（售后省心）。核心价值不变：**决策权永远在客户手里**。

### 1. 监管下自动回复·第一步已交付（"客户逐步授权"从路线图落地为现货）

之前这块整段是路线图预告，现在**已交付一个保守的第一步**，可以这样讲——但**严守只讲已交付的**：

- **三挡逐步授权，客户自己往上拨**（已交付）：自动沟通分**观察 / 监管 / 自治**三挡，挡位**由客户本人手动拨**，
  **系统绝不自作主张往上升**。话术：「不是甩手全自动——是客户**逐步授权**：先看它判得准不准（观察），
  信得过了让它发、自己旁看可一键刹车（监管），都是客户拨的，**决策权永远在客户手里**。」
- **群分层·真照顾客户不刷屏**（已交付）：**小群积极经营**（小群里主动回、维护关系），**大群只有@到我才回**
  （防止在大群里刷屏扰人）。话术：「它像个有分寸的数字销售员——**小群里积极经营客户关系，大群里绝不刷屏**，
  是真照顾客户、不打扰人。」
- **默认全关·客户安心起步**（已交付）：所有自动回复**出厂默认全部关闭**，由客户逐个会话亲手拨挡开启；
  且第一步**只自动发"看不看都无害"的安全话术**（如"收到""好的"），凡有分量的（承诺/金额/情绪）**一律交回人**。
  话术：「**默认一条都不自动发**，客户想开哪个会话、开到哪一挡，全是客户自己点的——起步零风险。」

> **⚠️ 不可超卖（销售必读）**：当前已交付的只是上面这"保守第一步"——**只自动发无害的安全话术、reply-only（只回不主动撩）**。
> 客户最想要的"让 AI 主动撩客户、自动展开有实质内容的对话"**仍是路线图（Phase 2），现在不可承诺、不可演示成现货**。
> 对客户口径：「**敢交给它**的那套护栏已经搭好、第一步已上线；让它**主动经营、深度对话**是下一批，逐步放开。」

### 2. 可观测·运维即服务（新卖点·售后省心）

面向"系统上线后谁来盯、出问题谁知道"的客户顾虑，新增一组**运维省心**卖点：

- **版本透明**：系统界面右下角永远显示当前版本号（如"AMR v0.11.0"），**客户和运维一眼知道跑的是不是最新版**，
  升级有据可查。话术：「不是个黑盒——它自己亮明身份和版本，升级、对账一目了然。」
- **数字运维工程师 7×24 监控告警**：系统开放一个**零隐私、无需口令**的运维健康接口，可以让**外部的数字运维工程师 Agent
  全天候盯着**——一旦自动发送被急停、后端掉线、发送失败堆积，**第一时间告警给人**。话术：「上线后不是没人管——
  有个数字运维工程师**7×24 盯着**，出问题先告警，**售后省心**。」
- **人在回路红线照旧**：监控 Agent **只看不动**——所有控制动作（急停、拨挡、确认发送）仍是**人的领地**，
  外部 Agent 碰不到客户隐私、也按不了任何发送按钮。话术：「监控的归监控、决策的归人——**省心但不失控**。」

> **三支柱进度**：关系智能、沟通教练持续加强；**监管下自动沟通迈出已交付的第一步**（保守、reply-only、默认全关）；
> **养心护体·保健医师仍为后续批次**。对外口径：已交付的讲现货，未交付的讲路线图，**人在回路·决策权在人**始终是当下成立的核心。

## 🔭 设计预告·下一批卖点（监管下自动沟通**深化**，**Phase 2 设计中·尚未交付，勿当现货卖**）

> 上节已交付"保守第一步"（reply-only、只发安全话术、默认全关）。**下面是 Phase 2 的深化方向**，
> 对客户**只可作"路线图预告"**讲，**不可承诺当前可用**——尤其"主动撩客户 / 自动展开实质对话"仍未交付。

> 设计正本：`docs/superpowers/specs/2026-06-28-supervised-agentic-router-design.md`。
> 以下是**已定稿的设计方向**，对客户**只可作"路线图预告"**讲，**不可承诺当前可用**。

- **三层护栏，让"AI 自动沟通"敢交给它**（客户最大的怕：AI 乱发 / 发错 / 半夜骚扰客户）。AMR 的纵深防御一句话讲清：
  - **得体靠时间窗** —— 绝不在不合适的点发，晚上不打扰客户。
  - **安全靠 LLM 分层** —— 只有"看不看都无害"的才自动发；有分量的（承诺 / 金额 / 情绪）**永远等你拍板**。
  - **兜底靠否决** —— 自动发前给你倒计时，一键否决。
  - 话术：「让 AI 在你监管下、上班时段主动撩客户，**三层护栏保证它不乱发、不越界、不半夜扰人，你随时能夺回**。」
- **安全区你自己划·话术库你自己灌**（特性 / 差异点）：AMR 给你一个界面，把**你的具体业务、要过的内容、你认可的话术**灌进去、训练进去；系统就**按"你的话术"安全自动沟通**——不是千人一面的客服腔，是你的生意、你的话。话术：「**安全圈是你自己画的**：把你的业务话术灌进去，AI 只在圈里自动、圈外交回你——越用越懂你的生意。」
- **AI-native 卖点·事的生命周期不腐烂**：传统 CRM 业务记录是死的，人一忙就烂在那被忘掉；AMR 把每件事的生命周期做成**引擎**，事闲置了 / 承诺到期了**主动冒出来要你处理**。话术：「把'会烂掉的业务'变成'监管下自我推进的线程'，没有一单被忘掉。」
- **★ 你逐步授权·决策权永远在你（AI 时代人不被替代）**：自动沟通分**观察 → 监管 → 自治**三挡，**由你自己往上拨**——先「观察」(它只摆出'本来会发啥'、一条不发)，信得过了拨「监管」(它发、你旁看可刹车)，再放心了拨「自治」(上班时段它自己跑，你每 1-2 小时回来看简报)。**挡是你拨的，系统绝不自作主张升**。话术：「**AI Agent 时代，人不但没被替代，反而更值钱**——你做创造和决策这些有价值的事，数字员工替你跑腿、放大、提效；但**决策权永远在你手里、你想给多少自治你拨多少**。AMR 守的就是这条：人在回路、人是决策者。」
  - 销售要主动讲给客户、让客户自己明白：**这套不是'甩手全自动'，是'你始终在掌舵、把杂活交给数字员工'**。客户不可把决策权交出去、不可偷懒——这既是底线、也是客户的价值所在。
- **人本节奏·真照顾客户不刷屏**：自动沟通的频率**模仿一个真人上班能照顾的量**(每天几十条要思考的消息、有限客户数)，不是机器群发。话术：「它像个尽职的数字销售员替你照顾客户，**节奏跟真人一样、不刷屏不扰人**——真照顾客户、满足需求，是在替你积德、做长久生意。」

> 纪律：本节随该特性真正交付后，再升格进正式卖点 + 同步 bump 版本头；交付前对客户一律"路线图"口径。**但「人在回路·决策权在人」是 AMR 当下就成立的核心价值(现在一切发送已是人审)，可直接讲，非路线图。**
